# Python

ialah satu bahasa pengaturcaraan (programming language) komputer yang sangat popular, ringkas, dan digunakan secara meluas di seluruh dunia.

# Tujuan

Ia direka supaya mudah dibaca dan ditulis oleh manusia kerana kodnya menggunakan perkataan bahasa Inggeris yang mudah, berbeza dengan bahasa pengaturcaraan lain yang kelihatan lebih rumit.

# Ciri-ciri Utama

- Sintaks yang mudah dan bersih
- Percuma dan sumber terbuka (open-source)
- Sokongan pelbagai platform (Windows, Mac, Linux)
- Komuniti yang besar dan aktif
- Boleh digunakan untuk pelbagai jenis projek

# Kegunaan

- Pembangunan web (Django, Flask)
- Sains data & Kecerdasan Buatan / AI (NumPy, Pandas, TensorFlow)
- Automasi & skrip
- Pembangunan permainan
- Pengaturcaraan saintifik dan matematik

# Versi Terkini

Python 3.x adalah versi yang disyorkan pada masa kini. Pastikan anda menggunakan versi terbaru untuk mendapat ciri dan keselamatan terbaik.

# Pemasangan

Muat turun Python secara percuma di laman rasmi:
https://www.python.org/downloads/

